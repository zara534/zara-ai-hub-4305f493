# ZARA AI HUB

An AI-powered platform for text generation, image creation, and smart conversations.

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase

## Development

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Deployment

This project is ready to deploy on Netlify or Vercel.
